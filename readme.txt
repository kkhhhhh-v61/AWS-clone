Eversummer Florist Website
==========================

A website for a florist company using HTML, CSS, PHP, JavaScript, and MySQL.

Features
--------

*   User registration and login system
*   Shopping cart system
*   Payment gateway (PayPal)
*   Admin panel to manage users, orders, and products
*   User account management (edit profile, change password, order history)
*   Product management (add, edit, delete)
*   Order management (view, delete, update status)
*   Customer behavior analysis

How to use
------------

1.  Download or clone the repository to your server.
2.  Create a database and import the `eversummer.sql` file.
3.  Update the `config/connect.php` file with your database credentials.
4.  Update the `paypal.php` file with your PayPal API credentials.
5.  Update the `admin/config/connect.php` file with your database credentials.
